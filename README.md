Movie Discovery App
A modern, responsive web application designed to help users explore, search, and discover the latest movies. Built using React and Vite, this platform integrates with The Movie Database (TMDB) API to provide real-time movie data.

🚀 Live Demo
https://movie-discovery-app-26.netlify.app/

🛠 Tech Stack
Frontend: React.js

Build Tool: Vite

API: The Movie Database (TMDB) API

Deployment: Netlify

🔑 Key Features
Trending Movies: View the latest and most popular movies directly on the homepage.

Search Functionality: Easily find specific movies by title with a dynamic search bar.

Responsive UI: Clean, modern interface designed to work seamlessly across mobile, tablet, and desktop devices.

Dynamic Data Fetching: Fetches up-to-date movie metadata, posters, and information by integrating with the TMDB API.
